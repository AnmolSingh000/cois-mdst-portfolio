Course Details

  • Description & Prerequisite

    Multimedia elements like photo and video editing, visual effects, 3D assets, and writing styles are key tools for exploring concepts and challenges in modern user experience design, particularly in web pages, game design, and user interfaces.

    Design principles focus on feature discoverability, understanding diverse users, managing feedback complexity, and how computer systems interpret and respond to user actions.

    Recommended Prerequisite: COIS 1010H

  • Learning Outcomes

    • Create and manipulate images using editing software.
    • Demonstrate basic HTML/CSS to construct a web page.
    • Design user interfaces for multiple platforms.
    • Analyze and discuss UX issues in software design.
    • Recognize legal frameworks for creative works.
    • Demonstrate foundational knowledge of gameplay & systems design.

  • Texts & Readings

    • Learning Web Design (5th ed) by Jennifer Robbins
    • The Design of Everyday Things by Donald Norman
    • Don't Make Me Think by Steve Krug

    External resources will be posted on Blackboard by the instructor.
